# P2 — Offline USDA packs

## Offline Essentials

Build a compact, versioned SQLite pack from Foundation Foods, FNDDS, and possibly SR Legacy. Support resumable download, integrity verification, atomic replacement, deletion, and a visible dataset date.

## Complete Offline

Investigate a separately downloadable Branded Foods pack only after measuring the normalized database and FTS index size. Provide a strict “never send food searches off this device” mode when an adequate pack is installed.
